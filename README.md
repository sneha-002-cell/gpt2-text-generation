# gpt2-text-generation

## 📌 Overview
This project demonstrates text generation using the GPT-2 transformer model. GPT-2 is a pre-trained language model capable of generating coherent and contextually relevant text.

## 🎯 Objective
To explore transformer-based language models and generate human-like text from prompts.

## ⚙️ Technologies
- Python
- Hugging Face Transformers
- GPT-2

## 🚀 Execution
Run:
python text_generation_gpt2.py
